# Guia Turístico - Presidente Prudente/SP

Aplicativo mobile simples desenvolvido em **React Native** com **Expo** e **React Navigation**, criado como atividade acadêmica para apresentar um guia turístico básico da cidade de **Presidente Prudente - SP**.

## Sobre o projeto

O aplicativo possui uma tela inicial com menu principal e permite navegar entre três categorias:

- Pontos Históricos
- Restaurantes Recomendados
- Atividades ao Ar Livre

Cada tela apresenta informações estáticas com descrições curtas sobre locais e atividades da cidade.

## Tecnologias utilizadas

- React Native
- Expo
- JavaScript
- React Navigation Stack
- VS Code
- Emulador Android Studio

## Funcionalidades

- Tela inicial com menu de navegação
- Navegação entre telas usando Stack Navigator
- Tela de pontos históricos da cidade
- Tela de restaurantes recomendados
- Tela de atividades ao ar livre
- Botão de voltar automático pelo cabeçalho do Stack Navigator
